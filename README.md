## :pushpin: How to start

```
// install mongodb and mongosh and node.js first
// install packages
npm i 

// start mongodb
"C:\Program Files\MongoDB\Server\6.0\bin\mongod.exe" --dbpath="c:\data\db"

// in the project directory
node app.js
```
